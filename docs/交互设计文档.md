# 尘智 (EmberInterDebugTool) — 交互设计文档

> 版本: v1.0  
> 日期: 2026-06-10  
> 基于 PRD: v3.0  
> 平台: Qt6 QML Desktop (Windows / Linux / macOS)

---

## 设计原则

| 原则 | 描述 |
|------|------|
| **开发者原生** | 遵循 VS Code / iTerm2 / SecureCRT 等专业工具的交互范式，零学习成本 |
| **人机可感知** | AI (CLI) 操作在 GUI 中可见（TX 标记），人在 GUI 的操作在 CLI 日志流可见 |
| **即时反馈** | 所有操作 <100ms 内给出视觉响应，无操作"静默丢失" |
| **容错优先** | 输入错误不清空，连接失败保留参数，配置损坏降级默认 |
| **空间高效** | 侧边栏可折叠，工具栏紧凑，不浪费纵向空间 |

## L1 — 概念级：信息架构与导航模型

### 1.1 应用外壳 (App Shell)

```
┌─────────────────────────────────────────────────────────┐
│  [App Icon] 尘智 — EmberInterDebugTool  [─][□][×]      │  ← 标题栏 (系统原生)
├────────┬────────────────────────────────────────────────┤
│        │  TabBar                                        │  ← 水平标签页栏
│        ├────────────────────────────────────────────────┤
│Sidebar │  Tab 内容区                                    │
│        │  (日志 / 终端 / 连接向导 / 设置)                 │
│        │                                                │
├────────┴────────────────────────────────────────────────┤
│  StatusBar                                              │  ← 状态栏
└─────────────────────────────────────────────────────────┘
```

**核心布局**: 左侧固定侧边栏 + 右侧动态内容区，底部全局状态栏。

### 1.2 导航模型

```
                    ┌──────────┐
                    │  主窗口   │
                    └────┬─────┘
                         │
          ┌──────────────┼──────────────┐
          │              │              │
     ┌────┴────┐   ┌────┴────┐   ┌────┴────┐
     │ 侧边栏   │   │ TabBar  │   │ 状态栏   │
     │ (固定)   │   │ (动态)   │   │ (固定)   │
     └────┬────┘   └────┬────┘   └─────────┘
          │              │
    ┌─────┴─────┐  ┌────┴─────────────┐
    │ 会话列表   │  │ StackLayout      │
    │ [+新建]   │  │  ├ SerialTab     │
    │ ST/JL入口 │  │  ├ TerminalTab   │
    │ 设置/关于 │  │  └ DebugTab      │
    └───────────┘  └──────────────────┘
```

**导航规则:**
1. 侧边栏始终可见（宽度 220px，可折叠至 48px 图标模式）
2. TabBar 按会话创建顺序排列，支持拖拽排序（v1 缺省）
3. 单击侧边栏会话 → 打开/切换到对应 Tab
4. 关闭 Tab → 断开连接，若为最后一个 Tab 则显示空状态页
5. ConnectionWizard 作为模态弹窗（非 Tab），创建成功后自动关闭并添加 Tab

### 1.3 用户流程

#### 流程 A: 创建串口会话

```
[点击侧边栏 +] → [弹出连接向导] → [选择"串口"页]
  → [填写名称] → [选择串口号(自动侦测,已占用置灰)]
  → [配置波特率/数据位/校验/停止] → [点击"创建会话"]
  → [向导关闭] → [侧边栏新增条目] → [自动连接] → [打开 SerialTab]
```

#### 流程 B: CLI Agent 连接

```
[CLI: --connect COM3] → [IPC消息到GUI]
  → [GUI创建SerialTab] → [TabBar新增标签]
  → [侧边栏高亮对应会话] → [CLI收到statusChanged]
  → [CLI --send "AT" ] → [GUI日志显示 [TX] AT (绿色)]
```

#### 流程 C: 终端会话 (本地/SSH)

```
[点击侧边栏 +] → [弹出连接向导] → [选择"终端"或"SSH"页]
  → [终端: 选择shell类型] / [SSH: 填写host/user/port/password]
  → [点击"创建会话"] → [PTY启动] → [TerminalView渲染]
  → [shell退出] → [Tab自动关闭]
```

### 1.4 页面清单

| 页面 | 类型 | 触发方式 | 关闭方式 |
|------|------|----------|----------|
| 主窗口 (空状态) | 根页面 | 应用启动 | 关闭窗口 |
| SerialTab | Tab 内嵌 | 连接串口会话 | 关闭 Tab / 断开 |
| TerminalTab | Tab 内嵌 | 连接终端/SSH会话 | 关闭 Tab / shell退出 |
| ConnectionWizard | 模态弹窗 | 侧边栏 [+] / CLI connect | 创建/取消/ESC |
| SettingsDialog | 模态弹窗 | 菜单/按钮 | 保存/取消/ESC |
| AboutDialog | 模态弹窗 | 菜单/按钮 | 关闭/ESC |

### 3.3 日志列表 (LogView / ListView)

**行渲染规格:**

```
┌──────────────────────────────────────────────────────────┐
│ 10:23:45.123  [INFO]  Device initialized                 │
│ ^^^^^^^^^^^^  ^^^^^^  ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ │
│ 时间戳(灰)    级别标签  文本内容(级别对应色)                 │
└──────────────────────────────────────────────────────────┘
```

**HEX 模式行渲染:**

```
┌──────────────────────────────────────────────────────────┐
│ 10:23:45.123  0000  48 65 6C 6C 6F 20 57 6F  72 6C 64   │Hello World│
└──────────────────────────────────────────────────────────┘
```

**交互行为:**
- 自动滚动: 新日志到达 → 自动滚动到底部
- 手动滚动: 用户向上滚动 → 暂停自动滚动, 右下角出现 [↓ 新消息] 浮动按钮
- 暂停模式: 点击 ⏸ → 日志继续缓冲但不渲染, 恢复后显示缓冲内容
- 过滤: 输入关键字 → ListView model 实时筛选, 匹配文字高亮 (黄色背景)
- 选中: 点击行选中 → 高亮行 (半透明蓝), 支持 Ctrl+C 复制文本
- 右键菜单: 复制行 / 复制全部 / 导出选中

**性能约束:**
- ListView 内置虚拟化 (cacheBuffer: 2000)
- 单次追加 >500 条时, 合并为批量 append, 避免逐条渲染

### 3.4 发送面板 (SendPanel)

**状态:**

| 状态 | 输入框 | 发送按钮 | 换行符选择 |
|------|--------|----------|-----------|
| 未连接 | disabled (灰底) | disabled (灰色) | disabled |
| 已连接 | enabled (深色底) | enabled (主题色) | enabled |
| 发送中 | enabled | loading (旋转图标) | enabled |
| 发送完成 | enabled | 短暂闪烁 (300ms 绿色) | enabled |

**发送逻辑:**
- 回车: 发送文本 (单行模式下)
- Ctrl+Enter: 始终发送
- 发送后: 输入框不自动清空 (保留用户输入)
- 发送历史: 存储在 ConfigManager, ComboBox 下拉可选
- HEX 输入: 空格分隔 `01 02 03` 或连续 `010203` 均可

### 3.5 过滤器 (FilterBar)

**交互:**
- 输入即过滤 (debounce 100ms)
- 左侧搜索图标 (始终显示)
- 右侧清除按钮 (有内容时显示)
- 过滤无结果: 列表中央显示 "无匹配结果"
- 过滤关键字持久化: 切换 Tab 后恢复各自过滤条件

| 状态 | 背景 | 边框 | 文字 |
|------|------|------|------|
| 未选中 | 透明 | 1px 对应色(串口蓝/终端绿/SSH橙) | 对应色 |
| 未选中+悬停 | 半透明对应色 | 1px 对应色 | 对应色 |
| 选中 | **对应色填充** | 1px 对应色 | **深色反色** |

### 3.9 终端颜色方案选择 (ColorScheme ComboBox)

**5 套预设:**

| 方案 | 前景 (fg) | 背景 (bg) | 适用场景 |
|------|-----------|-----------|----------|
| Catppuccin Dark | `#cdd6f4` | `#1e1e2e` | 默认，与整体UI一致 |
| Light | `#1e1e2e` | `#eff1f5` | 白天/亮环境 |
| Green Phosphor | `#00ff41` | `#0d0d0d` | 复古终端风格 |
| Amber | `#ffb000` | `#1a1a1a` | 复古琥珀屏 |
| Nord | `#d8dee9` | `#2e3440` | 低对比度, 护眼 |

**ComboBox 交互:**
- 下拉列表预览每条配色 (左侧色块 + 名称)
- 选中即时切换, 无过渡延迟

### 3.10 状态栏 (StatusBar)

```
● COM3 | 115200 8N1 | RX:12.3KB TX:456B | ↑ 02:34:12
```

**元素:**
- 连接指示器: 实心圆 (绿=已连接, 红=未连接, 黄=重连中)
- 当前端口: 文字
- 连接参数: `波特率 数据位校验位停止位`
- RX/TX 统计: 自动单位换算 (B → KB → MB)
- 运行时长: 自连接起计时
- 多个 Tab: 显示当前选中 Tab 的状态

**CLI 客户端计数:**
- 有 CLI 连接时, 状态栏右侧显示 "CLI:1" (灰色标签)

## L4 — 微交互：动画、反馈与边界处理

### 4.1 过渡动画规范

| 动画 | 时长 | 缓动 | 触发 |
|------|------|------|------|
| 按钮悬停色变 | 150ms | ease-out | 鼠标进入/离开 |
| 选项卡切换 (内容区) | 200ms | ease-in-out | TabButton 点击 |
| 向导页面滑动 | 250ms | ease-in-out | 类型按钮切换 |
| 弹出层出现 | 200ms | ease-out + fade | ConnectionWizard/对话框打开 |
| 弹出层消失 | 150ms | ease-in + fade | 关闭/ESC |
| 新日志行淡入 | 100ms | ease-out | 新 LogEntry 追加 |
| 连接指示脉冲 | 2000ms loop | ease-in-out | 已连接状态 |
| 光标闪烁 | 500ms on / 500ms off | 无 | 终端聚焦 |
| 侧边栏折叠/展开 | 250ms | ease-in-out | 折叠按钮点击 |
| Toast 滑入/滑出 | 300ms | ease-out | 操作反馈 |

### 4.2 连接状态指示器动画

**已连接 (绿):**
```
opacity: 1.0 ────────────────────────────────
           ╲    ╱    ╲    ╱    ╲    ╱
opacity: 0.6   ╲╱      ╲╱      ╲╱
           ← 2000ms loop →
```
- 呼吸脉冲: opacity 在 0.6~1.0 之间循环
- 表示数据正在传输 (非空闲)

**重连中 (黄):**
```
闪烁: 500ms on — 500ms off — 重复
```

**已断开 (红):**
```
常亮, 无动画
```

### 4.3 发送反馈

```
1. 用户点击 [发送]
2. 按钮短暂变化: 背景 → 绿色 (#22C55E), 图标 → ✓ (200ms)
3. 恢复正常 (300ms)
4. 日志区出现 [TX] 行 (淡入 100ms)
```

### 4.4 过滤无结果

```
┌──────────────────────────────────┐

│          ┌──────────┐            │

│          │ 无匹配结果 │            │

```

- 空状态图标居中显示
- 显示过滤关键字
- [清除过滤] 按钮一键恢复

### 4.5 连接错误处理

**串口连接失败:**

```
┌─────────────────────────────────────┐
│  ⚠ 连接失败                         │
│                                     │
│  无法打开 COM3: 拒绝访问              │
│  端口可能已被其他程序占用              │
│                                     │
│         [重试]  [更换端口]  [取消]     │
└─────────────────────────────────────┘
```

- 非阻塞 Toast/内联提示 (非模态弹窗)
- 自动重连时: 状态栏显示 "重连中... (第3次)" 不弹窗
- 连续失败 5 次: 停止自动重连, 显示错误提示

**SSH 连接失败:**

```
┌─────────────────────────────────────┐
│  ⚠ SSH 连接失败                     │
│                                     │
│  ssh: connect to host 192.168.1.100 │
│  port 22: Connection refused        │
│                                     │
│         [重试]  [编辑配置]  [取消]     │
└─────────────────────────────────────┘
```

### 4.6 CLI 操作在 GUI 的可见性

```
GUI 日志区的 TX 条目:
┌──────────────────────────────────────────────────────────┐
│ 10:23:46.001  [TX]    AT+GMR                    🤖 CLI   │
│                       ^^^^^^^                   ^^^^^^   │
│                       发送内容                   来源标记   │
└──────────────────────────────────────────────────────────┘
```

- CLI 发送的数据: 日志中显示 `[TX]` + 发送内容 + `🤖 CLI` 来源标记 (或 "Agent" 标签)
- 开发者手动发送: 日志中显示 `[TX]` + 发送内容 (无额外标记)
- CLI 连接/断开: 状态栏短暂提示 "CLI 客户端已连接/断开" (2s 后消失)

### 4.7 复制与选择

**终端区:**
- 拖拽选择文本 → 自动复制到剪贴板 (类 iTerm2)
- 右键菜单: 复制 / 粘贴 / 全选
- Ctrl+Shift+C / Ctrl+Shift+V (终端惯例)

**日志区:**
- 单击行选中 (高亮)
- Ctrl+C 复制选中行

- 右键菜单: 复制行 / 复制全部

|--------|--------|------|
| Ctrl+T | 全局 | 新建会话 (弹出连接向导) |

### 4.9 空状态与边界情况

| 场景 | 处理 |
|------|------|
| 无已保存会话 | 侧边栏 "我的会话" 下方显示空状态提示: "点击 [+] 创建第一个会话" |
| 无可用串口 | 串口 ComboBox 显示 "未检测到串口设备", 下拉为空 |
| 所有 Tab 关闭 | 主内容区显示空状态页 (Logo + 快速开始指南) |
| 终端 shell 退出 | Tab 自动关闭, 如为最后一个 Tab 显示空状态页 |
| 配置文件损坏 | 恢复默认配置, 启动后弹出一次性通知 "配置已重置为默认" |
| 日志缓冲区满 | 循环覆盖旧日志 (FIFO), 无提示 |
| GUI 最小化 | 串口/PTY 继续运行,IPC 继续服务 |
| 窗口缩放过小 | 最小尺寸 800×500, 低于此尺寸窗口不响应缩小 |

### 4.10 无障碍

| 需求 | 实现 |
|------|------|
| 键盘完整导航 | Tab/Shift+Tab 遍历控件, Enter/Space 激活 |
| 焦点可见 | 所有可交互控件有可见焦点环 (2px 主题色虚线) |
| 颜色非唯一标识 | 状态指示器同时使用图标+颜色, TX/RX 颜色配合文字标签 |
| 高对比度 | 主题色满足 WCAG AA (文本 4.5:1, 大文本 3:1) |

## 附：本次设计决策记录

| 决策 | 原因 |
|------|------|
| 侧边栏 220px + TabBar 结构 | VS Code / SecureCRT 行业惯例, 开发者零学习成本 |

| 自动滚动+浮动 [↓] 按钮 | 平衡实时查看与历史回溯, Safari/VS Code 已验证方案 |
| CLI 操作显示 🤖 来源标记 | 满足"人机可感知"核心需求, 让开发者知道哪些是 Agent 操作 |
